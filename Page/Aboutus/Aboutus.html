ABOUT RAILLY

Railly is a simple train-ticket web app that makes traveling easier. You can search routes,
check real-time seat availability, confirm your booking in one click,
and keep your e-tickets safely in one place.

The backend uses PHP + MySQL for secure bookings,
while the frontend is built with HTML, CSS, and JavaScript,
designed in Figma for a clean and responsive UI.
